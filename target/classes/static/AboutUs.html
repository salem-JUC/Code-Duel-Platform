CODE DUEL

About Us

Welcome to Code-Duel, a platform built by students, Where learning to code is a fun and competitive!

The Mission

help making coding engaging by combining learning with friendly competition. Whether you're a beginner just starting out or someone derusting their skills, Code-Duel hopefully helps you through hands-on practice.

What the platform Offer

- A growing library of programming challenges

- Real-time coding duels with your classmates or friends

- fast Automated code evaluation

- A leaderboard to track progress and improvement

Why We Built This

We noticed that most coding platforms were either too advanced or too boring for new learners. So we created Code-Duel as a class project — a lightweight, interactive environment that makes it easy to practice coding while having fun.
